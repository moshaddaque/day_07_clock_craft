# Screenshots Directory

Place your app screenshots in this directory to be used in the README.md file.

Recommended screenshots to include:

1. `digital_clock.png` - Screenshot of the digital clock interface
2. `analog_clock.png` - Screenshot of the analog clock interface
3. `widget_android.png` - Screenshot of the Android home screen widget
4. `widget_ios.png` - Screenshot of the iOS home screen widget

After adding your screenshots, update the URLs in the README.md file to point to your actual GitHub repository.
